# Areas and Volumes
Areas and volumes is a Python package that provides functions to calculate various geometric formulas. It supports 10 2D shapes and 10 3D shapes. Please dm me on discord for help or you want me to add more shapes. blackdoe846

# Variable meanings
2D Variables:

l = length

h = height

r = radius

sl = side length

b = base length


3D Variables:

l = length

w = width

h = height

r = radius

el = edge length

b = area of the base

MinR = Minor Radius

MajR = Major Radius


# Usage
2D shapes:

square(l)

rectangle(l, h)

circle(r)

triangle(b, h)

pentagon(sl)

hexagon(sl)

heptagon(sl)

octagon(sl)

nonagon(sl)

decagon(sl)

3D shapes:

cube(l)

rectangular_prism(l, w, h)

sphere(r)

cylinder(r, h)

cone(r, h)

dodecahedron(el)

triangular_prism(b, h)

hemisphere(r)

torus(MinR, MajR)

rhombicosidodecahedron(el)
